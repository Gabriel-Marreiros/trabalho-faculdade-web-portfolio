Sobre mim

Meu nome é Gabriel, sou desenvolvedor de software com 2 anos de experiência e sou apaixonado por transformar ideias e conceitos em soluções tecnológicas que impactam positivamente empresas e pessoas.

Estou ansioso para continuar minha evolução como programador, abraçando novos desafios e explorando novas possibilidades no mundo da tecnologia.

Portfólio

Abaixo estão alguns projetos que eu desenvolvi e que ficarão disponíveis em breve.

Web Portfólio

Este projeto foi criado para mostrar a minha trajetória profissional e educacional, apresentar os meus projetos de estudo e freelancer, meus certificados de cursos, as tecnologias e ferramentas nas quais possuo conhecimento e os meus contatos.

Este projeto foi desenvolvido utilizando as tecnologias: React, Typescript, Sass, Bootstrap e Docker

Help Desk Angular & Java

O Help Desk Angular & Java é uma aplicação criada para gerenciar a criação e atribuição de chamados. Além de possibilitar a criação, exclusão e atualização de usuários com diferentes perfis, como: técnico, cliente e administrador. Os clientes podem abrir chamados para os técnicos e os administradores conseguem inativar tecnicos e clientes.

Este projeto foi desenvolvido utilizando as tecnologias: Angular, Typescript, Sass, Bootstrap, Java, Spring Boot, MySQL e Docker

Inventory Management Angular & Java

Inventory Management Angular & Java é uma aplicação destinada ao gerenciamento de um inventário de produtos. A aplicação possui dois perfis de usuário, sendo eles administrador e usuário. Os administradores conseguem adicionar, editar e inativar produtos e categorias além de criar novos usuários, editar e inativar usuários existentes. Já o usuário comum consegue apenas criar e visualizar os produtos do inventário.

Este projeto foi desenvolvido utilizando as tecnologias: Angular, Typescript, Sass, Bootstrap, Java, Spring Boot, MySQL e Docker
